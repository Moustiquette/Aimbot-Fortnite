# Aimbot-Fortnite

Pour utiliser l'aimbot fortnite vous devez ouvrir le .exe Aimbot puis un système d'installation va s'ouvrir, lancer fortnite et appuyer sur inject, le cheat se lancera sur votre fortnite et il sera près a l'utilisation 


To use the fortnite aimbot you need to open the Aimbot .exe and then an installation system will open, launch fortnite and press inject, the cheat will launch on your fortnite and it will be ready to use
